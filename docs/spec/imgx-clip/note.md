# imgX-Clip コンテキストノート

**作成日**: 2026-03-11

## プロジェクト概要

画像のY軸方向を指定範囲でクリップするデスクトップアプリケーション。

## 技術スタック

- **フレームワーク**: Tauri v2
- **フロントエンド**: React + TypeScript
- **画像処理**: Rust (image crate) / Canvas API
- **対象OS**: Windows 11

## 開発ルール

- 特に追加ルールなし（新規プロジェクト）

## 関連実装

- 既存コードなし（新規プロジェクト）

## 設計上の注意事項

- ドラッグ操作のUXを重視（Tauri選定の理由）
- リアルタイム拡大プレビュー機能が必須
- 保存形式は入力と同じ形式を維持
- 軽量な単体exeとして配布可能にする
