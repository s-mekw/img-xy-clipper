## タスク概要

ファイル読み込み・保存ボタンを含むToolbarコンポーネントと、Appコンポーネントの状態管理（useReducer）を実装する。Toolbarからファイルダイアログを開き、IPCコマンドを呼び出す連携ロジックを含む。

- [ ] Toolbarに「ファイルを開く」「保存」ボタンがある
- [ ] Appコンポーネントで全体の状態管理（useReducer）が機能する
- [ ] 画像未読込・読込中・表示中・保存中・エラー等の状態遷移が正しい

## 主要実装項目

### Toolbarコンポーネント 🟡

**信頼性**: 🟡 *アーキテクチャ設計のコンポーネント構成から妥当な推測*

- 「ファイルを開く」ボタン → Tauriファイルダイアログ → `load_image` IPC呼び出し
- 「保存」ボタン → Tauri保存ダイアログ → `clip_and_save` IPC呼び出し
- 状態に応じたボタンの有効/無効制御

**実装ファイル**: `src/components/Toolbar.tsx`

### App状態管理（useReducer） 🟡

**信頼性**: 🟡 *データフロー図の状態管理から妥当な推測*

- AppState インターフェース（imagePath, imageData, imageWidth, imageHeight, imageFormat, clipTopY, clipBottomY, status, errorMessage）
- アクション: LOAD_START, LOAD_SUCCESS, LOAD_ERROR, SAVE_START, SAVE_SUCCESS, SAVE_ERROR, UPDATE_CLIP_REGION
- 状態遷移: idle → loading → ready → saving → ready / error

**実装ファイル**: `src/App.tsx`

### レイアウト・スタイル 🟡

**信頼性**: 🟡 *アーキテクチャ設計から妥当な推測*

- Toolbar上部、ImageCanvas左側、PreviewPanel右側のレイアウト
- 基本的なCSS設定

**実装ファイル**: `src/styles/index.css`
