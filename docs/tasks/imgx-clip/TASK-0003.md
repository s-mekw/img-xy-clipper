## タスク概要

Rust側で `clip_and_save` IPCコマンドを実装する。元画像のファイルパス、Y軸の開始・終了座標、保存先パスを受け取り、image crateで画像をクロップして入力と同じ形式で保存する。

## 主要実装項目

### clip_and_save IPCコマンド 🔵

**信頼性**: 🔵 *アーキテクチャ設計のIPCコマンド定義より*

- 入力: `{ src_path: String, top_y: u32, bottom_y: u32, dest_path: String }`
- 出力: 成功/失敗（Result型）
- image crateのcrop機能でY軸範囲をクロップ → 元の形式でエンコード → 保存

**実装ファイル**: `src-tauri/src/commands.rs`

### クロップ処理 🔵

**信頼性**: 🔵 *データフロー図・アーキテクチャ設計より*

- image crateの `crop_imm` でY軸範囲を切り取り
- X軸は画像全幅を維持
- 元画像の形式（PNG/JPG）を判定し同じ形式で保存

**実装ファイル**: `src-tauri/src/image_processor.rs`
