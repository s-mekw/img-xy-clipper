## タスク概要

Rust側で `load_image` IPCコマンドを実装する。ファイルパスを受け取り、image crateで画像を読み込み、Base64エンコードしたデータと画像メタデータ（幅・高さ・形式）をフロントエンドに返す。

**信頼性**: 🔵 *アーキテクチャ設計のIPCコマンド定義より*
